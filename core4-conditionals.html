Conditionals

Definition

If-then statements where specified actions only occur if certain conditions are met.

Scratch

Python

if user_input == "rest":
print(rest)
done = True
else:
user_input = input("Please type 'rest' or 'go on': ");

C++

if ((wLeft == 0) && (wRight == 0))
{
backward(1500);
leftServo.writeMicroseconds(1000);
rightServo.writeMicroseconds(1000);
delay(1500);
}
else
{
forward(20);
}
